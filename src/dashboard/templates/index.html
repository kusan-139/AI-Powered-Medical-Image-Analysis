🔬 Analysis Tasks

Three clinically-relevant computer vision tasks implemented end-to-end.

🫁

Pneumonia Detection

Binary classification on chest X-rays using MobileNetV2 Transfer Learning. Detects bacterial and viral pneumonia vs. normal lungs.

MobileNetV2 Binary Clf 5,863 imgs
🔬

Skin Lesion Classification

7-class classification across dermoscopy images including melanoma, nevus, and basal cell carcinoma using EfficientNetB0.

EfficientNet 7-class 10,015 imgs
🧠

Brain Tumour Segmentation

Pixel-wise semantic segmentation of brain MRI scans using U-Net with Dice Loss — localises and delineates tumour regions.

U-Net Segmentation Dice Loss
